1. Asegura el hook plymouth en mkinitcpio.conf

sudo nano /etc/mkinitcpio.conf

Busca la línea HOOKS= y déjala así:

HOOKS=(base udev plymouth autodetect microcode modconf kms keyboard keymap consolefont block filesystems fsck)

Guarda y regenera el initramfs:

sudo mkinitcpio -P

2. Editar la entrada normal de systemd-boot

sudo nano /boot/loader/entries/"nombre de tu archivo"

Cambios clave:

Añadimos initrd /intel-ucode.img (microcode de Intel).

En la línea de options añadimos quiet splash al final.

4. (Opcional) loader.conf un poco mejor
Si quieres menú y dejar marcada esa entrada por defecto:
sudo nano /boot/loader/loader.conf
Y lo dejas:

default 2025-11-18_04-31-22_linux.conf (aca va el nombre de tu archivo)
timeout 3
console-mode keep
